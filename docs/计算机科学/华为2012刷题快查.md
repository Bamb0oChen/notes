## NLP

1. LSTM (Long Short-Term Memory) 
原理：LSTM 是一种循环神经网络（RNN，Recurrent Neural Network），专门设计用于处理序列数据。其核心是“门控机制”（遗忘门、输入门、输出门），能够有效地捕获长距离的依赖关系，并在一定程度上解决传统 RNN 的梯度消失问题。

适用性：它在生成文本时，会维护一个“隐层状态”，相当于一个动态更新的记忆，能够根据之前生成的所有单词来决定下一个单词是什么。这使得它能够生成语法通顺、且在几十到几百个 token 范围内保持主题连贯的文本。

1. 隐马尔可夫模型 (HMM) 
原理：HMM 假设当前的隐藏状态仅依赖于前一个隐藏状态（一阶马尔可夫性），且当前的观测（单词）仅依赖于当前的隐藏状态（词性等）。

局限性：虽然 HMM 在词性标注等任务中表现出色，但在生成长文本时，它很难捕捉远超 1~2 步的复杂依赖关系。HMM 通常用于“给定模型找最优隐藏状态序列”，而不是用于自回归地生成自由文本。即便用于生成，其“上下文窗口”非常有限，文本会很快发散，缺乏长程连贯性。

3. 最大熵模型 (MaxEnt) 
原理：这是一种分类模型，用于在给定上下文的情况下预测下一个单词的概率分布。它不拘泥于生成式假设，可以融合多种特征。

局限性：最大的问题是无状态。在处理长文本时，它只能依赖一个固定长度的 N-gram 窗口（比如只看前 3 个单词）。一旦文本长度超过窗口大小，它就会丢失更早的历史信息。因此，生成的句子可能在局部是正确的，但段落整体逻辑会断裂。

4. 决策树 (Decision Tree) 
原理：决策树是一种树状结构的判别模型，通常用于分类或回归。

局限性：决策树无法直接生成文本。文本生成本质上是自回归的序列决策问题，而决策树是为静态特征设计的，无法处理变长的、顺序依赖的序列数据。
