## 1.1 1. 基本物理量

| 物理量 | 符号 | 定义/表达式 | 矢量/标量 | 单位(SI) | 关键注意点 |
|--------|------|--------------|------------|----------|--------------|
| 参考系 | - | 参照物 + 坐标系 + 时钟 | - | - | 惯性系中物理规律形式相同 |
| 质点 | $m$ | 有质量无大小的理想模型 | 标量 | kg | 忽略形状、转动、形变 |
| 位矢 | $\vec{r}(t)$ | 从原点指向质点的有向线段 | 矢量 | m | 随时间变化，$\vec{r} = x\hat{i}+y\hat{j}+z\hat{k}$ |
| 位移 | $\Delta \vec{r}$ | $\vec{r}_2 - \vec{r}_1$ | 矢量 | m | 与路径无关，只与初末位置有关 |
| 路程 | $\Delta s$ | 轨迹实际长度 | 标量 | m | $\Delta s \ge |\Delta \vec{r}|$，折返或曲线时取大于号 |
| 平均速度 | $\bar{\vec{v}}$ | $\frac{\Delta \vec{r}}{\Delta t}$ | 矢量 | m/s | 方向与$\Delta\vec{r}$相同 |
| 平均速率 | $\bar{v}$ | $\frac{\Delta s}{\Delta t}$ | 标量 | m/s | $\bar{v} \ge |\bar{\vec{v}}|$ |
| 瞬时速度 | $\vec{v}(t)$ | $\frac{d\vec{r}}{dt}$ | 矢量 | m/s | 方向为轨迹切线方向 |
| 瞬时速率 | $v(t)$ | $|\vec{v}(t)| = \frac{ds}{dt}$ | 标量 | m/s | 等于速度的大小 |
| 平均加速度 | $\bar{\vec{a}}$ | $\frac{\Delta \vec{v}}{\Delta t}$ | 矢量 | m/s² | 方向与$\Delta\vec{v}$相同 |
| 瞬时加速度 | $\vec{a}(t)$ | $\frac{d\vec{v}}{dt} = \frac{d^2\vec{r}}{dt^2}$ | 矢量 | m/s² | 方向不一定与$\vec{v}$相同 |

## 1.2 2. 常用坐标系及位矢表达式

| 坐标系 | 坐标变量 | 位矢表达式 | 适用场景 |
|--------|----------|--------------|----------|
| 笛卡尔(直角) | $(x,y,z)$ | $\vec{r} = x\hat{i} + y\hat{j} + z\hat{k}$ | 普遍适用，尤其直线运动、抛体运动 |
| 平面极坐标 | $(r,\theta)$ | $\vec{r} = r\hat{r}$ | 圆周运动、中心力场问题 |
| 柱坐标 | $(\rho,\phi,z)$ | $\vec{r} = \rho\hat{\rho} + z\hat{k}$ | 轴对称问题（如圆柱面运动） |
| 球坐标 | $(r,\theta,\phi)$ | $\vec{r} = r\hat{r}$ | 球对称问题（如万有引力、库仑场） |

## 1.3 3. 位移与路程对比

| 对比项 | 位移 $\Delta \vec{r}$ | 路程 $\Delta s$ |
|--------|----------------------|----------------|
| 类型 | 矢量 | 标量 |
| 依赖关系 | 仅初末位置 | 具体路径 |
| 大小比较 | $|\Delta\vec{r}|$ | $\Delta s \ge |\Delta\vec{r}|$ |
| 何时相等 | - | 单向直线运动 |
| 微元形式 | $d\vec{r}$ | $ds$（弧长） |
| 联系 | $|d\vec{r}| = ds$（瞬时） | $v = ds/dt$ |

## 1.4 4. 平均量与瞬时量对比

| 对比项 | 平均量 | 瞬时量 |
|--------|--------|--------|
| 速度 | $\bar{\vec{v}} = \Delta\vec{r}/\Delta t$ | $\vec{v} = d\vec{r}/dt$ |
| 速率 | $\bar{v} = \Delta s/\Delta t$ | $v = ds/dt$ |
| 加速度 | $\bar{\vec{a}} = \Delta\vec{v}/\Delta t$ | $\vec{a} = d\vec{v}/dt$ |
| 关系 | 极限 $\Delta t \to 0$ 得到瞬时量 | 瞬时量在时间上平均得到平均量 |

## 1.5 5. 常用运动学公式汇总

| 运动类型 | 已知条件 | 公式 |
|----------|----------|------|
| 匀速直线 | $v$ 常数 | $x = x_0 + vt$ |
| 匀加速直线 | $a$ 常数 | $v = v_0 + at$<br>$x = x_0 + v_0 t + \frac{1}{2}at^2$<br>$v^2 - v_0^2 = 2a(x-x_0)$ |
| 一般运动 | $\vec{r}(t)$ 已知 | $\vec{v} = d\vec{r}/dt$<br>$\vec{a} = d\vec{v}/dt$ |
| 一般运动 | $\vec{a}(t)$ 已知 + 初始条件 | $\vec{v} = \int \vec{a}\,dt + \vec{v}_0$<br>$\vec{r} = \int \vec{v}\,dt + \vec{r}_0$ |
| 变加速已知 $a(v)$ | $a = f(v)$ | 用 $\int dt = \int dv / f(v)$ 求 $v(t)$ |
| 变加速已知 $a(x)$ | $a = g(x)$ | 用 $v\,dv = a\,dx$ 求 $v(x)$ |

## 1.6 6. 微分方程构造

很多质点动力学/变加速题的关键，是先把“已知的物理量关系”翻译成可解的微分方程。最常用的出发点就三条（默认一维，用 $s(t)$ 表示路程/坐标）：

- 速度定义：$v=\dfrac{ds}{dt}$
- 加速度定义：$a=\dfrac{dv}{dt}=\dfrac{d^2s}{dt^2}$
- 链式法则（把二阶降为一阶）：$a=\dfrac{dv}{dt}=\dfrac{dv}{ds}\dfrac{ds}{dt}=v\dfrac{dv}{ds}$

| 已知信息（常见给法） | 选未知函数 | 直接写出的微分方程（构造结果） | 常见解法提示 |
|---|---|---|---|
| $a=a(t)$（加速度给成时间函数） | $v(t)$ 或 $s(t)$ | $\dfrac{dv}{dt}=a(t)$；或 $\dfrac{d^2s}{dt^2}=a(t)$ | 先积分得 $v(t)$，再积分得 $s(t)$；用初值定常数 |
| $v=v(t)$（速度给成时间函数） | $s(t)$ | $\dfrac{ds}{dt}=v(t)$ | 直接积分：$s(t)=\int v(t)\,dt + C$ |
| $a=a(v)$（加速度依赖速度，如阻力） | $v(t)$ | $\dfrac{dv}{dt}=a(v)$ | 常可分离：$\dfrac{dv}{a(v)}=dt$ |
| $a=a(s)$（加速度依赖位置，如弹簧/中心力的一维化） | $v(s)$ 或 $s(t)$ | $v\dfrac{dv}{ds}=a(s)$（等价于 $\dfrac{d^2s}{dt^2}=a(s)$） | 用“能量型”积分：$v\,dv=a(s)\,ds \Rightarrow \tfrac12 v^2=\int a(s)\,ds + C$ |
| $v=v(s)$（速度依赖位置） | $t(s)$ 或 $s(t)$ | $\dfrac{ds}{dt}=v(s)$ | 求时间更直接：$dt=\dfrac{ds}{v(s)}$，积分得 $t(s)$ |
| 受力已知 $F(\cdot)$（动力学表述） | $s(t)$ 或 $v(t)$ | 先写 $m\dfrac{dv}{dt}=F(\cdot)$ 或 $m\dfrac{d^2s}{dt^2}=F(\cdot)$ | 再看 $F$ 最终依赖 $t/s/v$，对应套用上面行 |

> 小提醒：微分方程通常需要初值/边值才能确定唯一解，例如 $s(t_0)=s_0,\ v(t_0)=v_0$。

## 1.7 7. 解题步骤速查

| 步骤 | 名称 | 操作内容 | 示例/提示 |
|------|------|----------|------------|
| 1 | 公式 | 根据已知量选择合适公式（定义式、运动方程） | 匀加速用三式，一般用微积分 |
| 2 | 代数 | 代入已知数值（符号或数字），注意符号方向 | 正负号代表方向 |
| 3 | 结果 | 写出所求物理量的表达式或数值 | 保留适当有效数字 |
| 4 | 单位 | 检查单位一致性，必要时转换为SI | 常见错误：cm、km未换算 |

## 1.8 8. 典型易错：相对绳速度 ≠ 对地速度

题型：两人质量均为 $m$，轻绳跨过光滑定滑轮；两人分别在两侧沿绳向上爬，且“甲相对绳的速率是乙的 2 倍”。问谁先到滑轮处。

关键点：轻绳 + 无摩擦滑轮 $\Rightarrow$ 绳中张力处处相等，记为 $T$。因此两人受力完全相同（对地）：上拉力 $T$、重力 $mg$。

取竖直向上为正，对每个人写牛顿第二定律（对地加速度）：
$$
T - mg = m a \quad\Rightarrow\quad a_A = a_B = \frac{T}{m} - g.
$$
若两人初速度相同，则任意时刻对地速度相同 $v_A(t)=v_B(t)$，两人高度差保持不变；若起点到滑轮的距离也相同，则到达时间相同（同时到达）。

“相对绳速率 2 倍”只是在确定绳子的运动：设左侧绳（甲所在侧）对地速度为 $u$（向上为正），右侧绳对地速度为 $-u$，则
$$
v_{A,\text{rel}}=v_A-u,\quad v_{B,\text{rel}}=v_B+u.
$$
结合 $v_{A,\text{rel}}=2v_{B,\text{rel}}$ 与 $v_A=v_B$ 可得 $u=-\tfrac{v_A}{3}$（绳向下运动），但并不改变“$a_A=a_B$、$v_A=v_B$、同时到达”的结论。
