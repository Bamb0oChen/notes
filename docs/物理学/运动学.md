## 1. 基本物理量

| 物理量 | 符号 | 定义/表达式 | 矢量/标量 | 单位(SI) | 关键注意点 |
|--------|------|--------------|------------|----------|--------------|
| 参考系 | - | 参照物 + 坐标系 + 时钟 | - | - | 惯性系中物理规律形式相同 |
| 质点 | $m$ | 有质量无大小的理想模型 | 标量 | kg | 忽略形状、转动、形变 |
| 位矢 | $\vec{r}(t)$ | 从原点指向质点的有向线段 | 矢量 | m | 随时间变化，$\vec{r} = x\hat{i}+y\hat{j}+z\hat{k}$ |
| 位移 | $\Delta \vec{r}$ | $\vec{r}_2 - \vec{r}_1$ | 矢量 | m | 与路径无关，只与初末位置有关 |
| 路程 | $\Delta s$ | 轨迹实际长度 | 标量 | m | $\Delta s \ge |\Delta \vec{r}|$，折返或曲线时取大于号 |
| 平均速度 | $\bar{\vec{v}}$ | $\frac{\Delta \vec{r}}{\Delta t}$ | 矢量 | m/s | 方向与$\Delta\vec{r}$相同 |
| 平均速率 | $\bar{v}$ | $\frac{\Delta s}{\Delta t}$ | 标量 | m/s | $\bar{v} \ge |\bar{\vec{v}}|$ |
| 瞬时速度 | $\vec{v}(t)$ | $\frac{d\vec{r}}{dt}$ | 矢量 | m/s | 方向为轨迹切线方向 |
| 瞬时速率 | $v(t)$ | $|\vec{v}(t)| = \frac{ds}{dt}$ | 标量 | m/s | 等于速度的大小 |
| 平均加速度 | $\bar{\vec{a}}$ | $\frac{\Delta \vec{v}}{\Delta t}$ | 矢量 | m/s² | 方向与$\Delta\vec{v}$相同 |
| 瞬时加速度 | $\vec{a}(t)$ | $\frac{d\vec{v}}{dt} = \frac{d^2\vec{r}}{dt^2}$ | 矢量 | m/s² | 方向不一定与$\vec{v}$相同 |

## 2. 常用坐标系及位矢表达式

| 坐标系 | 坐标变量 | 位矢表达式 | 适用场景 |
|--------|----------|--------------|----------|
| 笛卡尔(直角) | $(x,y,z)$ | $\vec{r} = x\hat{i} + y\hat{j} + z\hat{k}$ | 普遍适用，尤其直线运动、抛体运动 |
| 平面极坐标 | $(r,\theta)$ | $\vec{r} = r\hat{r}$ | 圆周运动、中心力场问题 |
| 柱坐标 | $(\rho,\phi,z)$ | $\vec{r} = \rho\hat{\rho} + z\hat{k}$ | 轴对称问题（如圆柱面运动） |
| 球坐标 | $(r,\theta,\phi)$ | $\vec{r} = r\hat{r}$ | 球对称问题（如万有引力、库仑场） |

## 3. 位移与路程对比

| 对比项 | 位移 $\Delta \vec{r}$ | 路程 $\Delta s$ |
|--------|----------------------|----------------|
| 类型 | 矢量 | 标量 |
| 依赖关系 | 仅初末位置 | 具体路径 |
| 大小比较 | $|\Delta\vec{r}|$ | $\Delta s \ge |\Delta\vec{r}|$ |
| 何时相等 | - | 单向直线运动 |
| 微元形式 | $d\vec{r}$ | $ds$（弧长） |
| 联系 | $|d\vec{r}| = ds$（瞬时） | $v = ds/dt$ |

## 4. 平均量与瞬时量对比

| 对比项 | 平均量 | 瞬时量 |
|--------|--------|--------|
| 速度 | $\bar{\vec{v}} = \Delta\vec{r}/\Delta t$ | $\vec{v} = d\vec{r}/dt$ |
| 速率 | $\bar{v} = \Delta s/\Delta t$ | $v = ds/dt$ |
| 加速度 | $\bar{\vec{a}} = \Delta\vec{v}/\Delta t$ | $\vec{a} = d\vec{v}/dt$ |
| 关系 | 极限 $\Delta t \to 0$ 得到瞬时量 | 瞬时量在时间上平均得到平均量 |

## 5. 常用运动学公式汇总

| 运动类型 | 已知条件 | 公式 |
|----------|----------|------|
| 匀速直线 | $v$ 常数 | $x = x_0 + vt$ |
| 匀加速直线 | $a$ 常数 | $v = v_0 + at$<br>$x = x_0 + v_0 t + \frac{1}{2}at^2$<br>$v^2 - v_0^2 = 2a(x-x_0)$ |
| 一般运动 | $\vec{r}(t)$ 已知 | $\vec{v} = d\vec{r}/dt$<br>$\vec{a} = d\vec{v}/dt$ |
| 一般运动 | $\vec{a}(t)$ 已知 + 初始条件 | $\vec{v} = \int \vec{a}\,dt + \vec{v}_0$<br>$\vec{r} = \int \vec{v}\,dt + \vec{r}_0$ |
| 变加速已知 $a(v)$ | $a = f(v)$ | 用 $\int dt = \int dv / f(v)$ 求 $v(t)$ |
| 变加速已知 $a(x)$ | $a = g(x)$ | 用 $v\,dv = a\,dx$ 求 $v(x)$ |

## 6. 解题步骤速查表

| 步骤 | 名称 | 操作内容 | 示例/提示 |
|------|------|----------|------------|
| 1 | 公式 | 根据已知量选择合适公式（定义式、运动方程） | 匀加速用三式，一般用微积分 |
| 2 | 代数 | 代入已知数值（符号或数字），注意符号方向 | 正负号代表方向 |
| 3 | 结果 | 写出所求物理量的表达式或数值 | 保留适当有效数字 |
| 4 | 单位 | 检查单位一致性，必要时转换为SI | 常见错误：cm、km未换算 |
