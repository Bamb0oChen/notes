## 精简批注（抓最能迁移的词块/句式）

### 段一（历史视角：先否定恐慌，再给规律）
`neither ... nor ...`：双重否定并列，开头很有“压住对方观点”的力量。

`peculiar to the present age`：`peculiar to` = “为……所特有”。可替换：`unique to`（更常用）。

`History, however, suggests ...`：`suggests` 用来“温和反驳”，比 `proves` 更稳。

`rarely ... altogether`：`altogether` = “完全地/彻底地”，常用于强调“并非彻底消失”。

`More often, ... rather than ...`：典型对照结构：多数情况下是 A，而不是 B。

### 段二（核心论证：别把效率当价值）
`is likely to follow the same course`：`follow the same course` = 走同样的轨迹（很书面）。

`Yet to infer from this that ... is to ...`：
	- 结构：`to infer ... is to ...` = “得出该结论等同于犯了某种错误”。
	- 可直接套用到写作：指出“把X当成Y”的谬误。

`mistake efficiency for value`：`mistake A for B` = 把 A 误当 B（高频高分）。

`Work is not defined solely by ...`：`not ... solely by` = 不应仅由……定义（议论文万能）。

`do more than ...`：表达“人类不止会做某事”，后面接更高层能力。

`interpret ambiguity / exercise judgment`：两组动宾搭配很地道，适合写“人类优势”。

`rather than a case`：`as a person rather than a case` 是“人本 vs 工具化”的经典对照。

`not ... a substitute for ..., but an extension of ...`：强对照模板，写科技题非常好用。

`promise lies not in ..., but in ...`：收束观点的金句框架（比 `the point is` 更高级）。

### 段三（升华：真正不可替代的是什么）
`in the end`：结论信号词。

`not A, but B`：最稳的对照句，适合结尾定性。

`excel at`：擅长于（比 `be good at` 更正式）。

`remain uniquely capable of`：`uniquely` 强调“独一无二的能力”。

`The real question ... is not whether ..., but whether ...`：
	- 这句是整篇最值得背的“转问句式”。
	- 用法：把讨论从“会不会”转到“我们准备好了没有”。

`another order of intelligence altogether`：`altogether` 再次强调“完全不同层级”。

`irreducibly human capacities`：`irreducibly` = 无法再被化简/拆解（文学性强）。更稳替换：`deeply human`。

`our lasting value resides`：`reside` = 位于/存在于（适合写抽象价值）。

`struggle to understand` / `be beyond it`：表达“AI 难以做到”的两种高级说法。

`To infer from X that Y is to mistake A for B.`

`The real question is not whether X will ..., but whether we are prepared to ...`
