## 一阶线性方程

一阶线性微分方程的标准形式为：

$$
\frac{dy}{dx} + P(x) \, y = Q(x).
$$

**“线性”** 的直观含义是：未知函数 $y$ 及其导数 $y'$ 在方程中只以 **一次幂** 的形式出现，且它们之间没有相乘（例如 $y^2$、$\sin y$、$y \cdot y'$ 等均不允许出现）。

如果把 $y'$ 和 $y$ 视为变量，方程形如：

$$
y' + P(x)\, y = Q(x),
$$

类似于一条“直线”方程 $aX + bY = c$，其中系数 $P(x), Q(x)$ 可以是 $x$ 的函数，但 $y$ 与 $y'$ 自身不能有非线性运算。

定义微分算子 $L$：

$$
L[y] = y' + P(x) \, y.
$$

如果算子 $L$ 满足以下两条性质，则称 $L$ 是 **线性算子**：

1. **可加性**：$L[y_1 + y_2] = L[y_1] + L[y_2]$；
2. **齐次性（数乘）**：$L[c\,y] = c\,L[y]$，其中 $c$ 为任意常数。

显然，$L[y] = y' + P(x)y$ 满足上述性质。因此方程

$$
L[y] = Q(x)
$$

被称为 **线性微分方程**。

特别地：

- 若 $Q(x) \equiv 0$，称为 **齐次线性方程**：$y' + P(x)y = 0$；
- 若 $Q(x) \not\equiv 0$，称为 **非齐次线性方程**：$y' + P(x)y = Q(x)$。

对于一阶线性非齐次方程

其通解可用 **常数变易法** 或 **积分因子法** 求得，结果为：

这种题目直接带公式

$$
y = e^{-\int p(x) \, dx} \left[ \int f(x) \, e^{\int p(x) \, dx} \, dx + C \right]
$$

其中 $C$ 为任意常数。

## 全微分方程（恰当方程）

形如$M(x,y)dx + N(x,y)dy = 0$如果存在$u(x,y)$满足全微分条件$du = \frac{\partial u}{\partial y}  = \frac{\partial u}{\partial y}dy$ 则通解为$u(x,y) = C$

1. 检验恰当性: $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$是否相等
2. 如果恰当，解$u(x,y)$
    1. 对M偏积分，得到$u(x,y) = \int M (x,y)dx + g(y)$其中g(y)待定
    2. 对上式求y的偏导，然后和N(y)比较求出g'(x)
    3. 求导得出g(x)最后的结果就是$u(x,y) = \int M (x,y)dx + g(y)$
3. 如果不是恰当的，加上积分因子
    1. 如果$\frac{M_y - N_x}{N} = f(x)$只和x有关，那么积分因子就是$\mu(x) = e^{\int f(x)dx}$则$\mu_y = 0$方程就是$\mu(M_y - N_x) = N\mu_y = N\frac{d\mu}{dx}$所以$\frac{d\mu}{\mu} = \frac{M_y - N_x}{N} = f(x)dx$然后积分

## 克莱罗方程

形式

$$y=xy'+f(y')$$

1. 令 p = y'
2. 对x求导，得到$y' = p+x\frac{dp}{dx} + f'(p)\frac{dp}{dx}$即$p = p+(x+f'(p))\frac{dp}{dx}$
3. 所以$(x+f'(p))\frac{dp}{dx} = 0$这样就只有两种可能
4. 解一：p=C
5. 解二：x=-f'(p)
