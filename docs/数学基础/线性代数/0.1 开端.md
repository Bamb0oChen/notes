行列式：
1.齐次：b1=b2=b3=...=bm=0
2.非齐次：b1=b2=b3=...=bm不全为0

**齐次一定有解**
1.唯一解（0解）
2.非零解（无数多解（乘系数k不变））

**非齐次**
1.无解
2.唯一解
3.无数多解
