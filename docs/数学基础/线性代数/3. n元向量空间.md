### 四个主要定理
1. 一个向量组线性相关，其中的一组向量必能有其他向量表示
2. 无关的线性向量组在添上新向量后相关，则添上的向量是原方程组的唯一表示
3. 多用少表示，则多相关
4. 一个无关向量能用另一个向量组表示，它的向量数小于另一个向量个数。
