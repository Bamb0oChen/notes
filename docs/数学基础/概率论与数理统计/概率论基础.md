## 1. 常见一元随机变量分布
| 分布名称 | 符号 | 概率分布律/密度函数 | 期望 $E(X)$ | 方差 $D(X)$ | 关键性质/备注 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **0-1 分布** | $X \sim 0-1(p)$ | $P(0)=1-p, P(1)=p$ | $p$ | $p(1-p)$ | 只有两种取值 |
| **二项分布** | $X \sim B(n,p)$ | $P(k)=C_n^k p^k (1-p)^{n-k}$ | $np$ | $np(1-p)$ | 独立考察 $n$ 次 |
| **泊松分布** | $X \sim \pi(\lambda)$ | $P(k)=\frac{\lambda^k}{k!}e^{-\lambda}$ | $\lambda$ | $\lambda$ | 记 $P(0), P(1), P(2)$ |
| **均匀分布** | $X \sim U(a,b)$ | $f(x)=1/(b-a), x\in(a,b)$ | $\frac{a+b}{2}$ | $\frac{(b-a)^2}{12}$ | 仅在区间内非零 |
| **正态分布** | $X \sim N(\mu,\sigma^2)$ | $f(x)=\frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$ | $\mu$ | $\sigma^2$ | $Z=\frac{X-\mu}{\sigma} \sim N(0,1)$ |
| **指数分布** | $X \sim E(\lambda)$ | $f(x)=\lambda e^{-\lambda x}, x>0$ | $1/\lambda$ | $1/\lambda^2$ | 无记忆性 |

## 2. 正态分布的重要性质
| 类别 | 内容 | 关键公式/结论 |
| :--- | :--- | :--- |
| **标准正态** | 分布函数 | $\Phi(-x) = 1 - \Phi(x)$ |
| | 概率计算 | $P(a < Z < b) = \Phi(b) - \Phi(a)$ |
| **二元正态** | 分解性质 | 若 $(X,Y) \sim N(\dots)$, 则边缘分布 $X, Y$ 仍为正态 |
| | 独立性 | 正态分布中：不相关 $\iff$ 独立 |

## 3. 二维随机变量（离散与连续）
| 概念 | 离散型公式 | 连续型公式 |
| :--- | :--- | :--- |
| **联合分布** | $p_{ij} = P(X=x_i, Y=y_j)$ | $f(x,y)$ |
| **边缘分布** | $p_i = \sum_j p_{ij}$ | $f_X(x) = \int_{-\infty}^{+\infty} f(x,y) dy$ |
| **条件分布** | $P(X\|Y) = p_{ij}/p_j$ | $f(y\|x) = f(x,y)/f_X(x)$ |
| **独立性判断**| $p_{ij} = p_i \cdot p_j$ | $f(x,y) = f_X(x) \cdot f_Y(y)$ |

## 4. 指数分布与特殊性质
| 性质名称 | 描述 |
| :--- | :--- |
| **无记忆性** | $P(X > a+b \mid X > a) = P(X > b)$ |
| **转化技巧** | $P(X \le x) = 1 - e^{-\lambda x}$ |

## 5. 解题技巧总结
* **泊松分布**：优先记忆 $P(0)=e^{-\lambda}, P(1)=\lambda e^{-\lambda}, P(2)=\frac{\lambda^2}{2}e^{-\lambda}$。
* **正态分布概率**：务必将事件转换为标准正态变量 $Z$ 的范围，再查表或写成 $\Phi$ 函数。
* **二元分布概率**：通过在非零区域 $D$ 上对联合密度函数 $f(x,y)$ 进行二重积分计算。
* **相关性判定**：计算 $\text{Cov}(X,Y) = E(XY) - E(X)E(Y)$。若 $>0$ 则正相关；若 $=0$ 则不相关。
