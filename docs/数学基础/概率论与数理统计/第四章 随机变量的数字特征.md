## 基本性质

期望在不独立的时候就具有了线性性，只有在独立的时候才具有结合性

同样，方差在不独立的时候就具有了线性性，其中我们的常数要做乘方处理

## 协方差
如果要说有什么新的部分，那很可能就是协方差了

协方差的本质是判断一个量影响另一个量的程度，所以在两个量独立的时候我们的协方差是0

### 协方差性质

1. **对称性**  
   $\text{Cov}(X, Y) = \text{Cov}(Y, X)$

2. **与方差的关系**  
   $\text{Cov}(X, X) = \text{Var}(X)$

3. **线性齐次性**  
   $\text{Cov}(aX, bY) = ab \cdot \text{Cov}(X, Y)$

4. **可加性**  
   $\text{Cov}(X_1 + X_2, Y) = \text{Cov}(X_1, Y) + \text{Cov}(X_2, Y)$

5. **独立时的性质**  
   若 $X$ 和 $Y$ 相互独立，则 $\text{Cov}(X, Y) = 0$

## 正态分布

另一个在概统中得到拓展的可能就是正态分布了(我们在后面会发现正态分布简直是上帝的分布)

### 1. 一元正态分布的性质

1. **标准正态分布** $Z \sim N(0,1)$ 的分布函数 $\Phi(x)$

   - 标准正态分布函数  
     $\Phi(x) = P(Z \leq x)$

   - 标准正态分布函数的对称性  
     $\Phi(-x) = 1 - \Phi(x)$

   - 区间概率计算  
     $P(a < Z < b) = \Phi(b) - \Phi(a)$

2. $\Phi(x)$ 的值通过手册查阅，考试会提供需要用到的 $\Phi(x)$ 值

3. **正态分布的线性函数依然是正态分布** → 可经线性函数转化为标准正态分布

   - 正态分布 → 标准正态分布的转化  
     $X \sim N(\mu, \sigma^2) \quad \Rightarrow \quad Z = \frac{X - \mu}{\sigma} \sim N(0,1)$

4. 因此求解正态分布问题，往往需要转化为标准正态分布处理

### 2. 二元正态分布的性质

1. **二元正态分布的 $X$ 和 $Y$ 各自依然遵循正态分布**

   - 二元正态分布的分解  
     $(X, Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho) \quad \Rightarrow \quad X \sim N(\mu_1, \sigma_1^2), \quad Y \sim N(\mu_2, \sigma_2^2)$

2. $X$ 和 $Y$ 的线性函数 $aX + bY + c$ 依然是正态分布

3. 此时新正态变量的 $\mu$ 和 $\sigma^2$ 分别是 $aX + bY + c$ 的均值和方差

4. **对于两个正态变量，不相关和独立是等价的**

5. 这使得判断正态变量是否独立时，只需计算协方差即可
