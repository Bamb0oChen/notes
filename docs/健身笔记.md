## 2）训练方式与饮食方式

| 模块 | 细分 | 执行方式 |
|---|---|---|
| 周一 | 胸 | 蝴蝶机夹胸热身 -> 卧推 -> 绳索下拉 -> 上斜卧推/哑铃卧推 |
| 周二 | 背 | 高位正手下拉 -> 高位反手下拉 -> 绳索面拉 -> 划船 -> 单臂划船 |
| 周三 | 腿 | 四头坐推?(那个机子我叫不上名) -> 夹腿 -> 深蹲 -> 罗马尼亚硬拉 -> 保加利亚深蹲 |
| 周四 | 肩 | 肩推 -> 哑铃推举/侧平举 -> 史密斯机(抢不到就加练手臂或胸) |
| 周五 | 臂 | 弯举 -> 臂屈伸 |


由于本人一般在饭后训练，所以一般不在训练前前加餐
|时间|补充方向|具体食物|
