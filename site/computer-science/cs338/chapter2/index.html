    transfromer 架构以及变体

    抱歉没有按照课程顺序编排，我只是感觉这一章值得记录，所以有了这篇文章

    Transformer变体组成参考文档

    一、梯度与反向传播基础

    梯度下降的核心公式

    新权重 = 旧权重 - 学习率 × 梯度 为什么是减号？ - 梯度指向损失函数增加最快的方向 - 我们的目标是减小损失，所以往反方向走

    前向与反向计算量对比

    方向 计算量 原因
    前向传播 O(n²) 一次矩阵乘法
    反向传播 O(2n²-3n²) 需要计算每个参数的梯度，应用链式法则

    二、多层网络与激活函数

    为什么需要隐藏层？

    网络类型 数学表达 拟合能力 直观比喻
    单层网络 y = wx + b 只能拟合直线 一把直尺
    多层网络（无激活） y = w₂(w₁x + b₁) + b₂ 仍是直线 几把直尺接起来，还是直的
    多层网络（有激活） y = w₂·激活(w₁x + b₁) + b₂ 可以拟合曲线 直尺 + 弯头积木

    激活函数的作用

    激活函数是神经网络中的"弯头积木"，让网络能够拐弯，从而拟合非线性关系。

    常见激活函数： - ReLU：max(0, x)——只保留正数部分 - GELU：x·Φ(x)——高斯误差线性单元，ReLU的平滑版本 - Swish：x·sigmoid(x)——自门控激活函数

    权重w₂的真实作用

    网络结构：x → [w₁] → h → [激活] → h' → [w₂] → y w₂的本质是一个放大器/衰减器： - w₂ > 1：放大信号 - 0 < w₂ < 1：缩小信号 - w₂ < 0：反向信号 - w₂ = 0：完全关闭通道

    关键认知： - ✅ w₂可以控制信号通过强度 - ❌ w₂不能创造非线性 - ✅ 非线性能力来自激活函数

    三、GLU（门控线性单元）

    从标准FF到GLU

    标准前馈网络（FF）： FF(x) = max(0, xW₁)W₂ GLU（门控线性单元）： GLU(x) = [max(0, xW₁) ⊗ (xV)] W₂

    GLU的核心创新

    GLU引入了一个额外的参数矩阵V，用来控制信息流的门控机制：

    • xW₁：主要内容（经过ReLU激活）
    • xV：门控信号（线性变换，无激活）
    • ⊗：元素级乘法，实现门控效果
    • 新参数V：门的控制器，让网络学会动态调节信息流量

    用生活案例理解GLU

    标准FF：不管三七二十一，直接执行 GLU： 先观察情况，再决定执行力度

    具体来说： • xV 就像是"情况判断器" • ⊗ 就像是一道"门" • 门开大（xV > 0且值大）→ 信息多通过 • 门关小（xV ≈ 0）→ 信息少通过 • 门反向（xV < 0）→ 信息极性反转

    GLU变体家族

    变体 公式 特点
    ReGLU max(0, xW₁) ⊗ (xV) ReLU激活 + 线性门
    GeGLU GELU(xW₁) ⊗ (xV) 高斯误差线性单元
    SwiGLU Swish(xW₁) ⊗ (xV) LLaMA等大模型采用
    GLU σ(xW₁) ⊗ (xV) Sigmoid做门控

    四、归一化策略

    两种归一化的数学形式

    后归一化（Post-Norm）——原始Transformer： output = LayerNorm( x + Attention(x) ) 预归一化（Pre-Norm）——现代大模型： output = x + Attention( LayerNorm(x) )

    为什么预归一化更优？

    对比维度 后归一化（Post-Norm） 预归一化（Pre-Norm）
    梯度传播 梯度需要穿过归一化层，容易消失 有残差"高速公路"，梯度直通
    数学形式 ∂out/∂x = 小量 ∂out/∂x = 1 + 小量 ✓
    训练稳定性 容易梯度爆炸，需要预热 稳定，学习率可设更大
    深层扩展 层数增加时浅层难以学习 可训练超深网络（GPT-3达96层）

    六、现代大模型组件选择

    推荐配置速查表

    组件 推荐选择 选择理由
    归一化位置 预归一化（Pre-Norm） 梯度不消失，训练稳定
    激活函数 GLU家族 门控机制增强表达能力
    GLU具体变体 ReGLU / SwiGLU 实验表现最佳，被广泛采用
    残差连接 必备 保证梯度流通

    主流大模型的实际选择

    模型 归一化策略 激活函数
    GPT-3 预归一化 GeLU
    LLaMA 预归一化 SwiGLU
    PaLM 预归一化 SwiGLU
    GLM-130B 预归一化 GeGLU
    Qwen 预归一化 SwiGLU
    Mistral 预归一化 SwiGLU

    七、核心公式汇总

    # 1. 梯度下降更新
θ_new = θ_old - η · ∇L(θ_old)

# 2. 预归一化Transformer块
x = x + Attention( LayerNorm(x) )
x = x + FF( LayerNorm(x) )

# 3. GLU前馈网络（以ReGLU为例）
FF_GLU(x) = [ max(0, xW₁) ⊗ (xV) ] W₂

# 4. 完整现代Transformer层
x_norm = LayerNorm(x)
attn_out = Attention(x_norm)
x = x + attn_out

x_norm = LayerNorm(x)  
ff_out = [ max(0, x_norm W₁) ⊗ (x_norm V) ] W₂
x = x + ff_out

    八、总结 Transformer演进的关键改进 1.预归一化：解决深层网络梯度消失问题

    2.GLU激活：引入门控机制，增强表达能力

    3.残差连接：保证梯度可以直通浅层

    设计现代Transformer的口诀

    预归一化打地基， 残差连接通梯度， GLU门控增能力， ReGLU/SwiGLU最给力！
